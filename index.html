About Quire

Hej och välkommen till Quire! Quire är en multifunktionell anteckningsbok som fungerar direkt i webbläsaren. Den första versionen av tjänsten sparar alla anteckningar lokalt i datorn/webbläsaren och har alltså ingen server eller backend. Som användare på Quire så kan du skapa flera olika anteckningar i din anteckningsbok.

Du kan i varje anteckning som du skapar skriva så mycket text som du önskar samt så kan du formatera denna fritt med enklare formatering såsom rubriker, punktlistor fetstil osv.

För anteckingar som är viktiga för dig så kan du favoritmarkera dom anteckningar för att kunna hitta dem enklare i din meny.

Ifall du skulle vilja så finns det inställningar kring val av typsnitt och färger för applikationen som helhet och kan på så sätt göra Quire till din personliga app.

Och självklart så kan du skriva ut dina anteckningar i pappers format :)

Tack för oss och njut av besöket!
